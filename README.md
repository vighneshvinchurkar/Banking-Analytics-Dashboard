# 📊 Banking Analytics Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Banking Analytics Dashboard developed in Microsoft Power BI to analyze customer accounts, balances, and transaction trends.

The dashboard enables users to explore banking data through dynamic visualizations and gain meaningful business insights.

---

## 🎯 Objectives

- Analyze customer transaction behavior
- Compare account balances
- Monitor monthly transaction trends
- Study customer demographics
- Visualize debit and credit distributions
